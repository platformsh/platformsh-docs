You might find the following commands useful when using the Symfony CLI.

-   Open the web administration console:

    ```bash
    symfony web
    ```

-   Open the URL of the current environment:

    ```bash
    symfony url
    ```

-   Open an SSH connection to your environment:

    ```bash
    symfony ssh
    ```
